[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's walk through our agent architecture.
The workflow starts at the START node and immediately goes into the security_checkpoint, which scrubs PII and checks for prompt injections.
If the security check fails, it routes to security_error_node and terminates.
If it passes, the workflow formats the input and invokes the orchestrator, our Lead Career Concierge.
The orchestrator delegates tasks to two specialized sub-agents: the job_analyzer, which extracts key details, and the candidate_fit_analyzer, which scores candidate fit.
From there, the resume_tailorer utilizes our local MCP server to search Canadian career resources and create compliant resume highlights.
Then, the cover_letter_generator drafts a cover letter in the job's primary language.
Next, the workflow enters a review_application node, which is a Human-in-the-Loop pause where the candidate reviews the drafts and provides feedback.
Once the candidate approves, the interview_coach generates custom interview prep questions and bilingual elevator pitches.
Finally, the generate_final_package node compiles the complete application.

Security is built directly into the fabric of ApplyWise AI.
